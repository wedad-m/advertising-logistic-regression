# Advertising Logistic Regression Assignment

This assignment applies Logistic Regression to advertising data to predict whether a user will click on an ad.

## Files
- `advertising_logistic_regression_assignment.ipynb` — Jupyter Notebook containing the full analysis, visualizations, model training, and evaluation
- `advertising.csv` — dataset used in the assignment

## Included Steps
- Data loading and exploration
- Data visualization
- Feature selection
- Train-test split
- Logistic Regression model training
- Prediction and evaluation

## Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
